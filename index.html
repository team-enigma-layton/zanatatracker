CORS 요청이 차단됨

브라우저에선 기본적으로 CORS(다른 도매인에 웹 요청을 하는 것)을 차단합니다. 이는 Zanata REST API 에 ZCT 가 접근하는 것을 막습니다.

차단을 해제하기 위해 확장기능을 설치하면 ZCT를 계속 이용하실 수 있습니다.
